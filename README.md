# Customer Relationship Management (CRM) System

A scalable CRM system designed to manage customer service operations, support requests, and internal workflows across multiple departments.

## Key Features
- Ticketing system with categorization and routing
- Multi-department workflow management
- Role-based access control (Admin, Manager, Team Leader, Agent)
- Customer profile and interaction tracking
- WhatsApp integration for notifications
- Service-based ticket classification and assignment

## Technologies
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS

## My Contribution
- Developed ticket routing and workflow engine
- Implemented role-based permissions and group management
- Integrated WhatsApp API for automated notifications
- Improved backend performance and data organization

## Availability
Private repository — Developed for a private company — Source code is not publicly available.

# نظام إدارة علاقات العملاء (CRM)

نظام CRM متكامل لإدارة خدمات العملاء وطلبات الدعم وتنظيم سير العمل بين الأقسام المختلفة.

## المميزات الرئيسية
- نظام تذاكر مع تصنيف وتوجيه تلقائي
- إدارة سير العمل بين الأقسام
- صلاحيات حسب الأدوار (مدير، مشرف، قائد فريق، موظف)
- إدارة بيانات العملاء وسجل التفاعلات
- تكامل مع واتساب للإشعارات
- تصنيف التذاكر حسب الخدمة

## التقنيات المستخدمة
- ASP.NET Web Forms
- C#
- SQL Server
- RDLC Reports
- JavaScript, Bootstrap 5
- HTML, CSS

## دوري في المشروع
- تطوير نظام التذاكر وتوجيه الطلبات
- بناء نظام الصلاحيات وإدارة المجموعات
- ربط النظام مع WhatsApp API للإشعارات
- تحسين الأداء وتنظيم البيانات
